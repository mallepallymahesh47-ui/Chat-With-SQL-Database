# Chat with SQL Database using AI

A simple AI-powered system that allows users to interact with SQL databases using natural language instead of writing queries.

## 🚀 Features
- Convert plain English → SQL queries
- Supports MySQL & SQLite
- Powered by Ollama (local AI)
- Fast and easy to use

## 🛠️ Tech Stack
- Python  
- LangChain
- Langsmith
- Ollama  
- SQLite / MySQL  
- Streamlit  

## 📌 How It Works
1. User enters a question in plain English  
2. AI converts it into an SQL query  
3. Query is executed on the database  
4. Results are returned in readable format
